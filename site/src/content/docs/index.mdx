import { Card, CardGrid } from '@astrojs/starlight/components';

Agents discover what they're allowed to access. Warden brokers every connection.
Operators get one control plane for **identity**, **policy**, and **audit** — across every
MCP server, cloud, code-host, observability stack, database, and SaaS the agent reaches.

<CardGrid>
  <Card title="Identity ✓" icon="approve-check">
    The agent presents only its own identity — a JWT, TLS client certificate, SPIFFE SVID, or
    Kubernetes service-account token. It never holds a credential.
  </Card>
  <Card title="Policy ✓" icon="approve-check">
    Every call is authorized the moment it happens — down to the action, its arguments, and on
    whose behalf the agent is acting.
  </Card>
  <Card title="Audit ✓" icon="approve-check">
    Every request is tied to the real identity behind it, the role used, the policy decision,
    and the upstream called.
  </Card>
</CardGrid>
